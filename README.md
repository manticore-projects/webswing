## How to Apply

```bash
# 1. Copy all files from this archive into your repo, preserving paths
#    (every file is relative to webswing-directdraw/)

# 2. Delete obsolete files
rm webswing-directdraw/webswing-directdraw-javascript/.babelrc
rm webswing-directdraw/webswing-directdraw-javascript/tslint.json

# 3. Update root pom.xml (see above)

# 4. Delete old node_modules and rebuild
cd webswing-directdraw/webswing-directdraw-javascript
rm -rf node_modules package-lock.json
npm install
npm run build

# 5. Verify
node -e "const m = require('./dist/main.js'); console.log(Object.keys(m));"
# Should print: [ 'DirectDraw', 'org' ]
```

## TypeScript Changes (diff)

Only type annotations — no behavioral changes:

- `new Promise(...)` → `new Promise<void>(...)` (4 places) — TS5 requires explicit void
- `new Promise(...)` → `new Promise<HTMLCanvasElement>(...)` (1 place) — explicit return type
- `catch (e) { onErrorMessage(e) }` → `onErrorMessage(e as Error)` (3 places) — TS5 unknown catch
- `Promise.all(...).then(resolve)` → `.then(() => resolve())` (2 places) — void compatibility
- `resolve: (value?: HTMLCanvasElement)` → `resolve: (value: HTMLCanvasElement)` — non-optional
